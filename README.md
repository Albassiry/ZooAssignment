# Zoo Assignment

## 📌 Description
This is a basic Java application for my assignment.  
It demonstrates object-oriented programming concepts using two classes: `Animal` and `Zoo`.

## 🐾 Classes
- **Animal.java**
  - Fields: `name`, `color`, `age`, `favFood`
  - Constructor to initialize all fields
  - Getters and setters for each field
  - `printDetails()` method to display animal information

- **Zoo.java**
  - Driver class
  - Uses `Scanner` to prompt the user for animal details
  - Creates an `Animal` object and prints its details

## ▶️ How to Compile and Run
1. Open terminal in the project folder.
2. Compile both files:
   ```bash
   javac Animal.java Zoo.java
